## Usage

1. Clone this repo

2. Install dependancies

   ```
   yarn
   ```

3. Review outdated dependancies and upgrade if needed

   ```
   yarn outdated
   ```

4. Run app

   ```
   yarn dev
   ```

   or if you have vercel-cli installed,

   ```
   vercel dev
   ```
